# 안티포렌식(Anti-Forensic)과 대응

## 1. 개요

### 가. 정의
> 디지털 포렌식 조사를 **방해·회피·무력화**하기 위해 증거를 은닉·훼손·변조하는 기술·행위.

### 나. 등장 배경
- 포렌식 수사 고도화에 대한 **역이용**, 범죄 은폐·내부정보 유출 은닉

## 2. 안티포렌식 기술

| 유형 | 기법 |
|---|---|
| **데이터 삭제·파괴** | 와이핑(Wiping), 물리 파괴 |
| **은닉** | 스테가노그래피, 슬랙공간·숨김파티션 |
| **암호화·난독화** | 볼륨 암호화, 패킹 |
| **변조·위조** | 타임스탬프 변경, 로그 삭제 |
| **흔적 제거** | 아티팩트·메타데이터 제거 |

## 3. 대응 컴플라이언스 시스템 — 구축 프로세스

```mermaid
flowchart LR
  P[정책·기준 수립] --> C[로그·증거 수집 체계]
  C --> I[무결성 보존<br/>해시·타임스탬프]
  I --> M[모니터링·통제]
```

- 정책 수립 → **로그·증거 수집 자동화** → 무결성 보존(해시·전자서명·WORM) → 접근통제·감사

## 4. 활용 프로세스
- **탐지**(이상행위·삭제 탐지) → **보존**(증거보전·CoC, Chain of Custody) → **분석·대응** → **재발방지·감사 보고**

---

> **한 줄 요약**: 안티포렌식은 *증거를 은닉·훼손·변조해 포렌식을 방해* 하는 기술이며, 컴플라이언스 시스템은 로그·증거 수집과 무결성 보존(해시·CoC)·모니터링으로 탐지-보존-분석-대응한다.

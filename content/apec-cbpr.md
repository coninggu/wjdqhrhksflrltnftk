# APEC CBPR (Cross Border Privacy Rules)

## 1. 개요

### 가. 정의
> APEC이 운영하는 **국경 간 개인정보 이전 인증체계**. 기업이 APEC 프라이버시 프레임워크 기준을 준수함을 인증받아 **회원국 간 안전한 개인정보 이전**을 지원.

### 나. 필요성
- 글로벌 데이터 이전의 **신뢰·상호운용성** 확보(GDPR 적정성과 병행)

## 2. APEC 프라이버시 9원칙

| # | 원칙 |
|---|---|
| 1 | **피해 예방**(Preventing Harm) |
| 2 | **고지**(Notice) |
| 3 | **수집 제한**(Collection Limitation) |
| 4 | **이용 제한**(Uses of Personal Information) |
| 5 | **선택**(Choice) |
| 6 | **완전성**(Integrity of Personal Information) |
| 7 | **보안**(Security Safeguards) |
| 8 | **접근·정정**(Access and Correction) |
| 9 | **책임성**(Accountability) |

## 3. CBPR 주요 인증기준

```mermaid
flowchart LR
  A[기업 신청] --> B[책임기관 심사<br/>Accountability Agent]
  B --> C[9원칙 기반 요건 평가]
  C --> D[인증·공개·사후관리]
```

| 영역 | 인증 기준(예) |
|---|---|
| **고지·선택** | 처리방침 고지, 동의·옵트아웃 |
| **수집·이용 제한** | 목적 내 수집·이용 |
| **보안** | 안전조치·접근통제 |
| **접근·정정** | 정보주체 권리 보장 |
| **책임성** | 이전 시 책임, 이행 점검·집행 |

## 4. 특징·비교

| 구분 | CBPR | GDPR 적정성 |
|---|---|---|
| **성격** | 기업 자율 인증 | 국가 단위 적정성 |
| **범위** | APEC 참여국 | EU |
| **집행** | 책임기관·프라이버시 집행기관 | 감독기구 |

## 5. 고려사항 및 시사점
- 글로벌 서비스의 **데이터 이전 신뢰 수단**, GDPR과 병행 대응
- Global CBPR Forum으로 확장, 국내 참여·인증 활성화 과제
- 개인정보보호법·국외이전 규정과 연계

---

> **한 줄 요약**: APEC CBPR은 *APEC 프라이버시 9원칙 기반의 기업 자율 국경간 개인정보 이전 인증체계* 로, 책임기관 심사로 고지·수집이용제한·보안·책임성 등 기준을 평가해 안전한 데이터 이전을 지원한다.

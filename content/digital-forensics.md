# 디지털 포렌식(Digital Forensics)

## 1. 개요

### 가. 디지털 포렌식의 개념 (1)
> 컴퓨터·모바일·네트워크 등 **디지털 기기에 저장된 데이터를 법적 증거로 수집·분석·보존**하는 과학 수사 기법. 범죄 수사·소송·사고 대응에 활용된다.

디지털 포렌식의 핵심은 '**증거능력(법정에서 인정받는 힘)**'이다. 아무리 중요한 데이터라도 수집·분석 과정이 부적절하면 증거로 인정받지 못한다. 그래서 원본 무결성(해시), 연계 보관성(Chain of Custody), 재현 가능성 같은 원칙을 엄격히 지켜야 한다.

### 나. 기본 원칙
| 원칙 | 내용 |
|---|---|
| **정당성** | 적법한 절차로 수집 |
| **무결성** | 원본 변경 없음(해시값 검증) |
| **연계 보관성** | 증거의 취득~분석 이력 추적(Chain of Custody) |
| **재현성** | 동일 결과 재현 가능 |
| **신속성** | 휘발성 증거 신속 확보 |

## 2. 유형과 절차 (2)

```mermaid
flowchart LR
  A[수집·보존] --> B[획득·이송] --> C[분석] --> D[보고·증거제출]
  style C fill:#e8f0fe,stroke:#2f6fed,stroke-width:2px
```

| 유형 | 대상 |
|---|---|
| **디스크 포렌식** | 저장매체(HDD·SSD) |
| **모바일 포렌식** | 스마트폰·태블릿 |
| **네트워크 포렌식** | 트래픽·로그 |
| **메모리 포렌식** | 휘발성 메모리(RAM) |
| **데이터베이스·클라우드** | DB·클라우드 저장 데이터 |

| 절차 | 내용 |
|---|---|
| **수집·보존** | 현장 증거 확보, 휘발성 우선 |
| **획득·이송** | 이미징(사본 생성)·해시 검증 |
| **분석** | 데이터 복원·타임라인·키워드 분석 |
| **보고** | 분석 결과 문서화·증거 제출 |

## 3. 기술 및 활용 분야 (3)

| 기술 | 내용 |
|---|---|
| **이미징·복제** | 원본 무결성 유지 사본 생성 |
| **데이터 복원** | 삭제·손상 데이터 복구 |
| **타임라인 분석** | 사건 시간순 재구성 |
| **안티포렌식 대응** | 은닉·삭제·암호화 우회 |

- **활용 분야**: 범죄 수사, 기업 내부감사·정보유출, 사고 대응(DFIR), 소송(e-Discovery)

## 4. 시사점
- 클라우드·모바일·IoT 확산으로 포렌식 대상·난이도 증가
- 안티포렌식·암호화 대응, AI 기반 대용량 분석 자동화
- 증거능력 확보를 위한 **절차 준수·도구 신뢰성**이 핵심

---

> **한 줄 요약**: 디지털 포렌식은 *디지털 증거를 무결성·연계보관성을 지켜 수집·분석·보존* 하는 과학 수사로, 디스크·모바일·네트워크·메모리 유형과 수집→획득→분석→보고 절차로 수행된다.

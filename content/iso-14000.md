# ISO 14000 (환경경영시스템)

## 1. 개요

### 가. 개념
> 조직의 **환경경영시스템(EMS)** 에 관한 국제표준 군(群). 핵심은 **ISO 14001**(EMS 요구사항, 인증 대상).

### 나. 필요성
- 환경규제·이해관계자 요구 대응, **지속가능경영·ESG** 실현
- 환경 리스크 관리, 자원·에너지 효율·기업 이미지 제고

## 2. 인증 규격(주요)

| 규격 | 내용 |
|---|---|
| **ISO 14001** | 환경경영시스템 요구사항(인증) |
| **ISO 14004** | EMS 구축 지침 |
| **ISO 14040/44** | 전과정평가(LCA) |
| **ISO 14064** | 온실가스 산정·검증 |

## 3. 구축 및 인증 절차

```mermaid
flowchart LR
  P[환경방침 수립] --> D[EMS 구축<br/>PDCA]
  D --> A[내부심사]
  A --> C[인증심사<br/>1·2단계]
  C --> R[인증서 발급·사후관리]
```

| 단계 | 내용 |
|---|---|
| **계획(P)** | 환경방침, 환경측면·법규 파악, 목표 설정 |
| **실행(D)** | 자원·역할·운영 통제·비상대응 |
| **점검(C)** | 모니터링·내부심사·부적합 시정 |
| **개선(A)** | 경영검토·지속개선 |
| **인증** | 1단계(문서)·2단계(현장) 심사 → 발급, **사후·갱신 심사** |

## 4. 인증 효과

| 효과 | 내용 |
|---|---|
| **법규 준수** | 환경규제 대응·리스크 감소 |
| **비용 절감** | 에너지·자원·폐기물 절감 |
| **신뢰·경쟁력** | 이미지 제고, 수출·거래 요건 충족 |
| **ESG** | 지속가능경영·공시 근거 |

## 5. 고려사항 및 시사점
- 인증 자체보다 **실질적 환경성과·지속 개선**이 중요
- ISO 50001(에너지)·ESG 공시와 통합 운영
- 그린워싱 방지 위한 데이터 신뢰성 확보

---

> **한 줄 요약**: ISO 14000은 *환경경영시스템(EMS) 국제표준* 으로 ISO 14001을 중심으로 PDCA 기반 구축·인증(1·2단계 심사)하며, 법규 준수·비용 절감·ESG 경쟁력 등의 효과를 제공한다.

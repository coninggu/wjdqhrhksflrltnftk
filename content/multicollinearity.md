# 다중공선성(Multicollinearity)

## 1. 개요

### 가. 정의
> 다중회귀에서 **독립변수들 간 강한 상관관계**가 존재해, 회귀계수 추정이 불안정해지는 현상.

### 나. 문제점
- 회귀계수의 **분산 팽창**(부호·크기 왜곡), 유의성 검정 신뢰 저하
- 개별 변수의 영향 해석 곤란(예측력은 유지될 수 있음)

## 2. 발생 원인

| 원인 | 예 |
|---|---|
| **중복 변수** | 키(cm)와 키(inch) 동시 투입 |
| **파생·합성 변수** | 합계·비율 변수 |
| **자연 상관** | 소득-소비, 면적-방 개수 |

## 3. 진단 방법

```mermaid
flowchart LR
  C[상관행렬] --> V[VIF 계산]
  V --> E[고유값·조건지수]
```

| 방법 | 판단 기준 |
|---|---|
| **상관계수** | 독립변수 간 상관 높음(±0.8↑) |
| **VIF(분산팽창계수)** | VIF ≥ 10 (또는 5) → 공선성 의심 |
| **공차(Tolerance)** | 1/VIF, 낮으면 문제 |
| **조건지수** | 30↑ 시 심각 |

## 4. 해결 방안

| 방안 | 내용 |
|---|---|
| **변수 제거** | 상관 높은 변수 중 하나 제거 |
| **변수 결합/차원축소** | 주성분분석(PCA), 요인분석 |
| **정규화 회귀** | Ridge(L2)·Lasso(L1)로 계수 안정화 |
| **데이터 보강** | 표본 확대, 중심화(centering) |

## 5. 고려사항 및 시사점
- 예측만 목적이면 공선성 허용 가능, **해석·추론 목적**엔 반드시 처리
- 머신러닝 특성 선택·규제(Regularization)와 연계
- 도메인 지식 기반 변수 설계로 사전 예방

---

> **한 줄 요약**: 다중공선성은 *독립변수 간 강한 상관으로 회귀계수 추정이 불안정* 해지는 현상으로, VIF(≥10)·조건지수로 진단하고 변수 제거·PCA·Ridge/Lasso로 해결한다.

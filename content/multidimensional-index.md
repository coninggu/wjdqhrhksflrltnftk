# 다차원 색인구조(Multidimensional Index Structure)

## 1. 개요

### 가. 정의
> 위치·공간·다속성 등 **2차원 이상의 다차원 데이터를 효율적으로 검색**하기 위한 색인 구조. 범위 질의·최근접 이웃(NN)·공간 질의를 빠르게 처리한다.

### 나. 필요성
- B-Tree 등 1차원 색인은 **다차원 범위·공간 질의에 비효율**
- 공간DB·이미지·위치 서비스의 급증

## 2. 유형

```mermaid
flowchart TB
  M[다차원 색인구조] --> T[트리 기반]
  M --> H[해시/그리드 기반]
  T --> T1["R-Tree / R*-Tree<br/>(MBR 계층)"]
  T --> T2["KD-Tree<br/>(공간 분할)"]
  T --> T3["Quad-Tree<br/>(사분 분할)"]
  H --> H1["Grid File<br/>(격자 분할)"]
  style M fill:#e8f0fe,stroke:#2f6fed,stroke-width:2px
```

| 유형 | 방식 | 특징 |
|---|---|---|
| **R-Tree / R*-Tree** | 최소경계사각형(MBR)으로 계층 구성 | 공간 객체·범위 질의, 상용 공간DB 표준 |
| **KD-Tree** | 축을 번갈아 이진 공간분할 | 점 데이터·NN 검색, 고차원서 성능 저하 |
| **Quad-Tree** | 공간을 사분면으로 재귀 분할 | 2D 영상·지도, 희소 데이터 |
| **Grid File** | 다차원 격자로 버킷 분할 | 균등 분포 데이터에 효율 |

## 3. 활용 사례

| 분야 | 활용 |
|---|---|
| **공간DB·GIS** | 지도·위치 기반 서비스(주변 검색, 영역 질의) |
| **멀티미디어 검색** | 이미지·특징벡터 유사도(NN) 검색 |
| **OLAP·다차원 분석** | 다차원 큐브 범위 집계 |
| **AI·벡터 검색** | 고차원 임베딩 근사최근접(ANN)의 기반 개념 |

## 4. 시사점
- **차원의 저주**: 고차원일수록 성능 저하 → 차원 축소·근사(ANN) 병행
- 데이터 분포·질의 유형(점/범위/NN)에 맞는 구조 선택
- 벡터 검색(HNSW·IVF)으로 진화

---

> **한 줄 요약**: 다차원 색인구조는 *R-Tree·KD-Tree·Quad-Tree·Grid File* 등으로 다차원·공간 데이터의 범위·NN 질의를 가속하며, GIS·멀티미디어·벡터 검색에 활용된다.

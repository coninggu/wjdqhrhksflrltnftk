# 인공신경망(Artificial Neural Network)

## 1. 개요

### 가. 개념·구성요소·역할
> 인간 뉴런을 모방해 **입력→가중합→활성화**로 패턴을 학습하는 모델.

| 구성요소 | 역할 |
|---|---|
| **뉴런(노드)** | 입력의 가중합에 활성화 함수 적용 |
| **가중치·편향** | 학습으로 조정되는 파라미터 |
| **계층(입력·은닉·출력)** | 특징 추출·변환·예측 |
| **활성화 함수** | 비선형성 부여 |

## 2. 피드포워드 신경망(FFNN)
> 신호가 **입력→은닉→출력** 한 방향으로만 전파되는 기본 신경망.

```mermaid
flowchart LR
  I[입력층] --> H[은닉층<br/>가중합+활성화]
  H --> O[출력층]
```

- 절차: 입력 → 각 층 **가중합(z=Wx+b)** → **활성화** → 출력(예측)

## 3. 역전파(Backpropagation)
> 출력 오차를 **뒤로 전파**하며 경사하강법으로 가중치를 갱신하는 학습 알고리즘.

| 절차 | 내용 |
|---|---|
| 1. 순전파 | 예측값 계산 |
| 2. 손실 계산 | 예측-정답 오차(Loss) |
| 3. 오차 역전파 | 연쇄법칙으로 각 가중치 **기울기** 산출 |
| 4. 가중치 갱신 | 경사하강법으로 조정(반복) |

## 4. 활성화 함수의 종류

| 함수 | 특징 |
|---|---|
| **Sigmoid** | 0~1, 기울기 소실 |
| **Tanh** | -1~1 |
| **ReLU** | 음수 0, 학습 빠름(주력) |
| **Leaky ReLU/GELU** | ReLU 개선 |
| **Softmax** | 다중분류 확률 출력 |

---

> **한 줄 요약**: 인공신경망은 *가중합+활성화* 뉴런의 계층 구조로, FFNN이 순방향 예측을 수행하고 역전파로 오차 기울기를 계산해 가중치를 갱신하며, ReLU·Softmax 등 활성화 함수로 비선형성을 부여한다.

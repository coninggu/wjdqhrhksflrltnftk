# 점추정(Point Estimation)과 구간추정(Interval Estimation)

## 1. 개요

### 가. 정의
> 표본으로부터 **모집단의 모수(평균·비율 등)를 추론**하는 통계적 추정 방법. **점추정**은 하나의 값으로, **구간추정**은 확률적 구간으로 모수를 추정한다.

### 나. 필요성
- 전수조사 불가 시 **표본으로 모집단 특성 추론**
- 추정의 **정확성·신뢰도** 표현 필요

## 2. 점추정

| 항목 | 내용 |
|---|---|
| **개념** | 모수를 **단일 값**으로 추정(예: 표본평균 x̄로 모평균 μ 추정) |
| **추정량 조건** | **불편성**(편의 없음)·**효율성**(분산 최소)·**일치성**·충분성 |
| **한계** | 오차·신뢰도 정보를 제공하지 못함 |

## 3. 구간추정

| 항목 | 내용 |
|---|---|
| **개념** | 모수가 포함될 **구간(신뢰구간)** 을 신뢰수준과 함께 추정 |
| **신뢰구간** | 추정값 ± (임계값 × 표준오차) |
| **신뢰수준** | 95%·99% — "동일 방법 반복 시 그 비율만큼 참값 포함" |
| **예** | μ의 95% CI = x̄ ± 1.96·(σ/√n) |

## 4. 비교

| 구분 | 점추정 | 구간추정 |
|---|---|---|
| **결과** | 단일 값 | 구간(하한~상한) |
| **정보** | 추정치만 | 추정치 + **신뢰도·오차** |
| **정밀도 표현** | 없음 | 신뢰수준·구간폭 |
| **관계** | 구간추정의 중심 | 점추정 + 오차한계 |

- 표본 크기 n↑ → 표준오차↓ → **구간폭 좁아짐**(정밀도↑)

## 5. 고려사항 및 시사점
- 구간추정이 **불확실성까지 표현**해 의사결정에 유용
- 정규성·표본크기·모분산 여부에 따라 z/t 분포 선택
- 데이터 분석·품질관리·AI 성능 신뢰구간에 활용

---

> **한 줄 요약**: 점추정은 *모수를 단일 값(불편·효율 추정량)으로*, 구간추정은 *신뢰수준과 함께 구간으로* 모수를 추정하며, 구간추정이 추정의 신뢰도·오차까지 표현해 더 많은 정보를 제공한다.

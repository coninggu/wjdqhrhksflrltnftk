# POP(Point Of Production, 생산시점관리)

## 1. 개요

### 가. 정의
> **POP(Point Of Production)** 는 생산 현장에서 발생하는 정보를 **발생 시점·발생 장소에서 실시간으로 수집·처리**하여 생산 관리에 활용하는 시스템. 판매 시점을 관리하는 POS의 생산 버전이다.

POP의 핵심 가치는 '**생산 현장의 실시간 가시성 확보**'다. 과거에는 작업자가 종이에 생산 실적을 적어 나중에 입력했기 때문에, 관리자는 지금 라인이 어떻게 돌아가는지, 어디서 불량이 나는지를 실시간으로 알 수 없었다. POP는 설비·작업자·자재에서 나오는 데이터(생산량·가동상태·불량·작업시간)를 발생하는 그 순간, 그 자리에서 자동 수집한다. 그러면 관리자는 현장을 실시간으로 파악해 즉시 대응하고, 정확한 데이터로 생산성·품질을 관리할 수 있다. POP는 현장(설비)과 상위 정보시스템(MES·ERP)을 잇는 다리 역할을 하며, 스마트 팩토리의 데이터 수집 기반이 된다.

### 나. 필요성
수기·사후 입력은 부정확하고 지연되어 실시간 관리가 불가능했다. 현장 데이터를 실시간으로 확보해 생산성·품질·납기를 개선하려는 요구가 POP를 낳았다.

POP는 현장 설비·작업자로부터 데이터를 수집하는 단말(터치스크린·바코드·센서)과, 이를 처리·전달하는 소프트웨어로 구성되어, 상위의 MES(제조실행시스템)·ERP와 연계된다. 현장의 최하위 데이터 수집 계층에 위치한다.

| 구성요소 | 역할 |
|---|---|
| **입력 장치** | 바코드·RFID·터치패널·센서로 데이터 수집 |
| **POP 단말·SW** | 실시간 수집·표시·1차 처리 |
| **상위 연계** | MES·ERP로 데이터 전달 |

## 3. 수집·활용 정보

| 구분 | 내용 |
|---|---|
| **생산 실적** | 생산량·양품/불량·진척 |
| **설비 상태** | 가동·정지·고장(가동률) |
| **작업 정보** | 작업자·작업시간·공정 |
| **품질** | 불량 유형·발생 지점 |

## 4. 고려사항 및 시사점

1. **MES·스마트팩토리의 데이터 기반**이다. POP가 정확한 현장 데이터를 실시간으로 공급해야 MES의 생산 관리와 상위 분석·의사결정이 신뢰성을 갖는다.
2. **IoT·센서와의 융합**으로 진화한다. 사람이 입력하던 방식에서 설비 센서·IoT로 자동 수집하는 방향으로 발전해, 데이터 정확도와 실시간성을 높인다.
3. **데이터 표준화·연계가 관건**이다. 이기종 설비의 데이터를 표준 형식으로 수집·연계해야 상위 시스템에서 통합 분석이 가능하므로, 프로토콜(OPC-UA 등) 표준화가 중요하다.

---

> **한 줄 요약**: POP는 *생산 현장 정보를 발생 시점·장소에서 실시간 수집* 하는 시스템으로, 설비·작업자 데이터를 자동 수집해 MES·ERP와 연계하며 스마트 팩토리의 실시간 가시성과 데이터 기반을 제공한다.

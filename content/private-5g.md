# 5G 특화망(Private 5G) 구축 시 고려사항

## 1. 개요

### 가. 정의
> 특정 기업·기관이 **자체 건물·공장 등 특정 구역에 전용으로 구축·운영**하는 5G 네트워크. 국내에서는 **이음5G**(2021~, 28GHz·4.7GHz 전용 주파수 할당)로 제도화.

### 나. 특징
- 5G 3대 특성: **eMBB**(초광대역)·**URLLC**(초저지연·고신뢰)·**mMTC**(대규모 연결)
- **로컬 코어·MEC**로 데이터 외부 유출 없이 저지연 처리
- 스마트팩토리·물류·의료·국방 등 **미션크리티컬** 산업 DX 인프라

## 2. 안정성·신뢰성 확보 방안

```mermaid
flowchart LR
  U[URLLC<br/>초저지연·고신뢰] --> R[이중화·리던던시]
  R --> E[엣지 코어/MEC]
  E --> Q[네트워크 슬라이싱·QoS]
  Q --> S[보안·격리]
```

| 방안 | 내용 |
|---|---|
| **URLLC 적용** | 1ms급 초저지연·99.999% 고신뢰 전송(재전송·짧은 TTI) |
| **이중화(리던던시)** | 코어·전송·전원·기지국 이중화로 SPOF 제거, 페일오버 |
| **엣지 컴퓨팅(MEC)** | 로컬 처리로 지연·백홀 의존 최소화, 연속성 확보 |
| **네트워크 슬라이싱** | 용도별 논리망 분리로 트래픽 격리·QoS 보장 |
| **보안** | 전용망 물리·논리 격리, 상호인증·암호화, 접근통제 |
| **모니터링** | 실시간 성능·장애 감시, SLA 관리 |

## 3. 간섭 회피 방안

```mermaid
flowchart LR
  F[주파수 계획·할당] --> B[빔포밍·소출력 셀]
  B --> I[간섭 조정 ICIC/eICIC]
  I --> M[전파 측정·차폐]
```

| 방안 | 내용 |
|---|---|
| **주파수 계획** | 채널 할당·재사용(Frequency Reuse), 인접 대역 간섭 최소화 |
| **빔포밍·소출력 셀** | 지향성 안테나·스몰셀로 셀 경계 간섭 저감 |
| **간섭 조정** | ICIC/eICIC, 전력 제어(Power Control), 스케줄링 조율 |
| **전파 환경 최적화** | 실내 전파 측정·시뮬레이션, 차폐·중계기 배치 |
| **주파수 공유** | 공용 대역 사용 시 LBT·조정으로 공존 |

## 4. 고려사항 및 시사점
- **비용 대비 효과**(전용 주파수·장비·운영), 통신사 협력·구축 모델(직접/위탁)
- OT 환경 특성상 **가용성·보안**이 최우선 → 이중화·격리 필수
- 스마트팩토리·자율물류·원격의료의 **디지털 전환 핵심 인프라**로 확대

---

> **한 줄 요약**: 5G 특화망(이음5G)은 *전용 5G로 초저지연·고신뢰* 를 제공하며, URLLC·이중화·MEC·슬라이싱으로 안정성·신뢰성을, 주파수 계획·빔포밍·간섭조정·전파 최적화로 간섭 회피를 확보한다.

# 5G 특화망(Private 5G) 구축 고려사항

## 1. 개요

### 가. 정의
> 특정 기업·기관이 **자체 구역에 구축·운영**하는 전용 5G 네트워크(국내 이음5G). 초저지연·고신뢰가 필요한 스마트팩토리·의료 등에 활용.

### 나. 특징
- **eMBB·URLLC·mMTC**, 전용 주파수(28GHz·4.7GHz), 자체 코어(로컬)

## 2. 안정성·신뢰성 확보 방안

```mermaid
flowchart LR
  R[이중화·리던던시] --> E[엣지 코어<br/>로컬 브레이크아웃]
  E --> Q[QoS·슬라이싱]
  Q --> S[보안·인증]
```

| 방안 | 내용 |
|---|---|
| **URLLC** | 초저지연·고신뢰 전송(1ms) |
| **이중화** | 코어·전송·전원 리던던시 |
| **엣지 컴퓨팅(MEC)** | 로컬 처리로 지연·신뢰성 개선 |
| **네트워크 슬라이싱** | 용도별 논리망 분리·QoS 보장 |
| **보안** | 전용망 격리, 상호인증·암호화 |

## 3. 간섭 회피 방안

| 방안 | 내용 |
|---|---|
| **주파수 계획** | 채널 할당·재사용, 인접 간섭 최소화 |
| **소출력·빔포밍** | 셀 설계·지향성 안테나로 간섭 저감 |
| **간섭 조정** | ICIC/eICIC, 전력 제어 |
| **차폐·측정** | 전파 측정·실내 설계 최적화 |

## 4. 시사점
- 스마트팩토리·물류·의료 등 **미션크리티컬 산업 DX**의 인프라 기반

---

> **한 줄 요약**: 5G 특화망은 *전용 5G로 초저지연·고신뢰* 를 제공하며, 이중화·MEC·슬라이싱으로 안정성을, 주파수 계획·빔포밍·간섭조정으로 간섭 회피를 확보한다.

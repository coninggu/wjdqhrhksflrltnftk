# 웹서버 보안 — 리버스 프록시와 DDoS 사이버대피소

## 1. 개요

### 가. 배경
> 웹서버를 안전하게 운영하기 위한 방안으로, 클라이언트와 웹서버 사이에서 완충·보호 역할을 하는 **리버스 프록시(Reverse Proxy)** 와, 대규모 DDoS 공격 시 트래픽을 우회·정화하는 **DDoS 사이버대피소** 를 다룬다.

웹서버 보안의 핵심 발상은 '**웹서버를 직접 노출하지 말고 앞단에 방어벽을 두라**'는 것이다. 웹서버가 인터넷에 직접 연결되면 공격 표면이 그대로 노출되고, 트래픽이 몰리면 쉽게 다운된다. 리버스 프록시는 웹서버 앞에 서서 모든 요청을 대신 받아 처리한 뒤 뒤의 실제 서버로 전달하는 중계자다. 이렇게 하면 실제 서버의 위치·구조가 숨겨지고(은닉), 부하가 분산되며, SSL 처리·캐싱·보안 필터링을 앞단에서 일괄 수행할 수 있다. 한편 DDoS는 수많은 좀비 PC로 트래픽을 폭주시켜 서비스를 마비시키는 공격인데, 개별 조직이 감당하기 어려운 대규모 공격은 **사이버대피소** 로 대응한다. 공격 트래픽을 대피소(정화 센터)로 우회시켜 악성 트래픽을 걸러낸(정화) 뒤 정상 트래픽만 서버로 보내는 것이다.

### 나. 필요성
웹서버는 가장 노출된 공격 대상이며 DDoS는 갈수록 대형화된다. 앞단 방어(리버스 프록시)와 대규모 공격 대응(사이버대피소)이 안정적 서비스 운영에 필수다.

## 2. 리버스 프록시

```mermaid
flowchart LR
  C[클라이언트] --> RP["리버스 프록시<br/>(중계·보호)"] --> W["웹서버<br/>(은닉)"]
  style RP fill:#e8f0fe,stroke:#2f6fed,stroke-width:2px
```

리버스 프록시는 클라이언트의 요청을 대신 받아 뒤의 웹서버로 전달하고 응답을 돌려준다(포워드 프록시가 클라이언트를 대리하는 것과 반대로, 서버를 대리한다). 설정은 프록시 서버(Nginx·HAProxy 등)를 웹서버 앞에 두고, 요청을 실제 서버로 라우팅하도록 구성한다.

| 기능 | 내용 |
|---|---|
| **서버 은닉** | 실제 서버 IP·구조 감춤(공격 표면 축소) |
| **부하 분산** | 여러 서버로 요청 분배(로드밸런싱) |
| **SSL 종료** | 암복호화를 앞단에서 처리 |
| **캐싱·압축** | 정적 콘텐츠 캐싱으로 성능↑ |
| **보안 필터링** | WAF 연계, 악성 요청 차단 |

## 3. DDoS 사이버대피소

```mermaid
flowchart LR
  A["공격 트래픽<br/>(DDoS)"] --> S["사이버대피소<br/>(트래픽 정화)"] --> W[웹서버]
  style S fill:#e8f0fe,stroke:#2f6fed,stroke-width:2px
```

사이버대피소는 대규모 DDoS 공격을 받는 조직의 트래픽을 우회시켜, 악성 트래픽을 걸러낸(스크러빙) 뒤 정상 트래픽만 원 서버로 전달하는 정부·통신사 등의 방어 서비스다. 개별 조직이 감당 못하는 대량 공격을 대신 흡수·정화한다.

| 구성 | 내용 |
|---|---|
| **트래픽 우회** | DNS·BGP로 대피소로 유입 |
| **트래픽 정화** | 악성(봇넷) 트래픽 필터링·차단 |
| **정상 전달** | 정제된 정상 트래픽만 서버로 |

## 4. 고려사항 및 시사점

1. **다층 방어의 조합**이 필요하다. 리버스 프록시(앞단 보호·부하분산), WAF(웹 공격 차단), 사이버대피소(대규모 DDoS)를 계층적으로 결합해 웹서버를 종합 방어한다.
2. **평시 대비와 신속 전환**이 관건이다. 사이버대피소는 공격 발생 시 신속히 트래픽을 우회해야 효과가 있으므로, 사전 연동·훈련이 필요하다.
3. **CDN·클라우드 기반 방어로 진화**한다. 대규모 CDN·클라우드가 리버스 프록시·DDoS 방어·WAF를 통합 제공해, 엣지에서 공격을 흡수·정화하는 방식이 확산되고 있다.

---

> **한 줄 요약**: 웹서버 보안은 *리버스 프록시로 서버를 은닉·보호(부하분산·SSL·캐싱·필터링)* 하고, 대규모 DDoS는 *사이버대피소로 트래픽을 우회·정화* 하며, WAF·CDN과 결합한 다층 방어로 안정적 운영을 실현한다.

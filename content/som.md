# SOM(Self Organizing Map, 자기조직화지도)

## 1. 개요

### 가. 정의 및 특징
> Kohonen(1982)이 제안한 **비지도 학습 신경망**으로, 고차원 입력 데이터를 **저차원(주로 2D) 격자에 위상(topology)을 보존하며 매핑**하는 군집·시각화 기법.

| 특징 | 내용 |
|---|---|
| **비지도 학습** | 정답 레이블 없이 데이터 구조 학습 |
| **위상 보존** | 입력 공간의 인접 관계를 출력 격자에 유지 |
| **경쟁 학습** | 뉴런 간 경쟁으로 승자(BMU) 선정 |
| **차원 축소·시각화** | 고차원을 2D로 군집·가시화 |

## 2. SOM 구성요소

```mermaid
flowchart LR
  I[입력층<br/>고차원 벡터] -->|가중치 연결| O[경쟁층/출력층<br/>2D 격자 뉴런]
  O --> B[BMU<br/>승자 뉴런]
  B --> N[이웃 뉴런<br/>가중치 갱신]
```

| 구성요소 | 내용 |
|---|---|
| **입력층** | 고차원 입력 벡터 |
| **경쟁층(출력층)** | 2D 격자 뉴런, 각 뉴런은 가중치 벡터 보유 |
| **가중치 벡터** | 입력과 비교되는 참조 벡터 |
| **BMU(Best Matching Unit)** | 입력과 가장 유사한 승자 뉴런 |
| **이웃 함수** | BMU 주변 뉴런을 함께 갱신(가우시안) |

## 3. 학습 절차

| 순서 | 내용 |
|---|---|
| 1 | 입력 벡터 제시 |
| 2 | **BMU 선정**(최소 거리 뉴런) |
| 3 | BMU와 이웃 가중치를 입력 방향으로 갱신 |
| 4 | 학습률·이웃 반경 축소하며 반복 |

## 4. SOM과 지도학습 신경망의 차이

| 구분 | SOM | MLP 등 |
|---|---|---|
| **학습 방식** | **비지도** | 주로 **지도** |
| **학습 규칙** | 경쟁 학습(승자독식) | 오차역전파 |
| **목적** | 군집·차원축소·시각화 | 분류·예측 |
| **출력** | 위상 보존 2D 지도 | 클래스·연속값 |
| **구조** | 입력-경쟁층(2층) | 다층 은닉층 |

## 5. 고려사항 및 시사점
- 고객 세분화·이상탐지·패턴 시각화 등에 활용
- 위상 보존으로 **데이터 구조의 직관적 해석** 가능
- K-means 대비 군집 간 관계(인접성) 표현 우수, 격자 크기·학습률 튜닝 필요

---

> **한 줄 요약**: SOM은 *경쟁 학습 기반 비지도 신경망* 으로 고차원 데이터를 위상 보존하며 2D로 매핑(군집·시각화)하며, 오차역전파 지도학습 신경망과 목적·학습규칙이 다르다.

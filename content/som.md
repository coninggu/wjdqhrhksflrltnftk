# SOM(Self Organizing Map, 자기조직화지도)

## 1. 개요

### 가. 정의
> Kohonen(1982)이 제안한 **비지도 학습 신경망**으로, 고차원 입력 데이터를 **저차원(주로 2D) 격자에 위상(topology)을 보존하며 매핑**하여 군집화·시각화하는 기법.

SOM의 핵심 발상은 인간 대뇌 피질의 작동 방식에서 왔다. 뇌에서 비슷한 감각 자극이 인접한 뉴런 영역에서 처리되듯, SOM도 **비슷한 입력이 출력 격자에서 가까운 뉴런에 대응**되도록 스스로 조직화한다. 그 결과 고차원 데이터의 복잡한 이웃 관계가 2차원 지도 위에 사람이 눈으로 볼 수 있는 형태로 펼쳐진다. 이 "위상 보존"이 SOM을 단순 군집화와 구별 짓는 결정적 특징이다.

### 나. 특징
SOM은 정답 레이블 없이 데이터의 내부 구조를 학습하는 비지도 방식이며, 뉴런들이 입력을 놓고 경쟁해 승자를 뽑는 **경쟁 학습**으로 동작한다. 특히 승자만 갱신하는 것이 아니라 **승자의 이웃까지 함께** 갱신하는 점이, 인접 뉴런이 비슷한 값을 갖게 만들어 위상 보존을 실현한다.

| 특징 | 내용 |
|---|---|
| 비지도 학습 | 정답 레이블 없이 데이터 구조 학습 |
| 위상 보존 | 입력 공간의 인접 관계를 출력 격자에 유지 |
| 경쟁 학습 | 뉴런 간 경쟁으로 승자(BMU) 선정 |
| 차원 축소·시각화 | 고차원을 2D 격자로 군집·가시화 |

## 2. SOM 구성요소

```mermaid
flowchart LR
  I[입력층<br/>고차원 벡터] -->|가중치 연결| O[경쟁층/출력층<br/>2D 격자 뉴런]
  O --> B[BMU<br/>승자 뉴런]
  B --> N[이웃 뉴런<br/>가중치 갱신]
```

SOM은 은닉층이 여러 겹인 일반 신경망과 달리 **입력층과 경쟁층(출력층) 단 두 층**으로 이뤄진 단순한 구조다. 입력층은 고차원 입력 벡터를 받고, 경쟁층은 2D 격자로 배열된 뉴런들로 구성되며 **각 뉴런은 입력과 같은 차원의 가중치(참조) 벡터**를 하나씩 가진다. 입력이 들어오면 모든 뉴런의 가중치 벡터와 거리를 비교해 가장 가까운 뉴런을 승자, 즉 **BMU(Best Matching Unit)** 로 뽑고, BMU와 그 이웃 뉴런들의 가중치를 입력 쪽으로 끌어당긴다. 이때 이웃을 얼마나 함께 당길지는 거리에 따라 감소하는 **이웃 함수(주로 가우시안)** 가 결정한다.

| 구성요소 | 내용 |
|---|---|
| 입력층 | 고차원 입력 벡터 |
| 경쟁층(출력층) | 2D 격자 뉴런, 각 뉴런은 가중치 벡터 보유 |
| 가중치 벡터 | 입력과 비교되는 참조 벡터 |
| BMU(Best Matching Unit) | 입력과 가장 유사한 승자 뉴런 |
| 이웃 함수 | BMU 주변 뉴런을 함께 갱신(가우시안) |

## 3. 학습 절차

학습의 핵심은 **"승자를 뽑고, 승자와 그 이웃을 입력 쪽으로 조금씩 이동시키는 것을 반복"** 하는 것이다. 중요한 점은 학습이 진행될수록 학습률과 이웃 반경을 **점차 축소**한다는 것인데, 초반에는 넓은 이웃을 크게 움직여 전체적인 지도의 큰 틀을 잡고(전역 정렬), 후반에는 좁은 이웃을 미세하게 조정해 세부 구조를 다듬는다(국소 미세조정). 이 점진적 수축이 안정적 수렴과 위상 보존을 동시에 가능하게 한다.

| 순서 | 내용 |
|---|---|
| 1 | 입력 벡터 제시 |
| 2 | 모든 뉴런과 거리 비교 → 최소 거리 뉴런을 BMU로 선정 |
| 3 | BMU와 이웃 가중치를 입력 방향으로 갱신 |
| 4 | 학습률·이웃 반경을 축소하며 반복 |

## 4. SOM과 지도학습 신경망의 차이

SOM과 MLP(다층 퍼셉트론) 등 일반 지도학습 신경망은 목적 자체가 다르다. MLP는 정답을 알려주고 **오차를 역전파**해 분류·예측을 학습하는 반면, SOM은 정답 없이 **경쟁 학습(승자독식)** 으로 데이터의 구조를 파악해 군집·시각화한다. 오차역전파가 없다는 점, 위상을 보존한 2D 지도를 출력한다는 점이 근본적 차이다.

| 구분 | SOM | MLP 등 지도학습 |
|---|---|---|
| 학습 방식 | 비지도 | 주로 지도 |
| 학습 규칙 | 경쟁 학습(승자독식) | 오차역전파 |
| 목적 | 군집·차원축소·시각화 | 분류·예측 |
| 출력 | 위상 보존 2D 지도 | 클래스·연속값 |
| 구조 | 입력-경쟁층(2층) | 다층 은닉층 |

## 5. 고려사항 및 시사점
SOM의 실무 가치는 **결과를 사람이 해석할 수 있다**는 데 있다. 예컨대 고객을 구매 패턴으로 세분화할 때, K-means는 군집 번호만 주지만 SOM은 유사한 고객군을 지도상 인접 영역에 배치해 **군집 간의 관계(인접성)** 까지 보여준다. 그래서 고객 세분화·신용 위험 시각화·이상탐지·패턴 분석에 유용하다. 다만 격자 크기·학습률·이웃 반경 등 하이퍼파라미터에 결과가 민감해 튜닝이 필요하고, 대규모·초고차원 데이터에서는 학습 비용과 해석의 한계가 있다. 기술사 관점에서는 SOM을 예측 모델의 대체재가 아니라, 모델링에 앞서 데이터 구조를 이해하고 시각적으로 소통하는 **탐색적 분석·설명 도구**로 자리매김하는 것이 적절하다.

---

> **한 줄 요약**: SOM은 *경쟁 학습 기반 비지도 신경망* 으로 고차원 데이터를 위상을 보존하며 2D 격자에 매핑(군집·시각화)하며, 승자(BMU)와 이웃을 함께 갱신하고 학습률·반경을 축소해 수렴시키는 점에서 오차역전파 지도학습 신경망과 목적·학습규칙이 다르다.
